# 메타 선택 모달
- 오른쪽 아래 플로팅 버튼을 통해 활성화
- 바깥 영역을 터치해 닫기
- 항상 바깥 영역을 터치할 수 있도록 높이 제한
- 화면 진입 시 `/jpaui/meta` 요청을 통해 데이타 목록 구성 (문자열 배열)
- 데이타를 선택하면 모달은 닫히고 매인 화면에 해당 데이타 관리 ui 구성

# 매인 화면
- 최초 진입 시 상단에 데이타 선택 가이드 문구 제공
- 데이타 선택 시
  - 상단에 선택된 데이타 표시
  - `/jpaui/{선택한 데이타}` 요청을 통해 아이템 목록 구성 (json 배열)
  - 아이템의 필드는 json의 구조를 통해 동적으로 라벨과 입력란 생성
  - 모든 아이템은 공통 필드 `n` 을 가지고 있으며 이는 수정 불가
  - 아이템을 10개 단위로 패이징하여 제공 (화면에서 계산)
  - 패이징 버튼은 현재 패이지 기준 +- 2개씩
- 아이템 제출 버튼을 누르면, 확인 모달을 통해 변경값을 보여줌
- 확인 모달 내 최종 제출 버튼 api 는 todo
